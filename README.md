# context-study

This repository is a project to study the new context api (React v16.3.2)

## Getting Started

### Installing

Running project:

* Clone the project:

* Install all packages via Yarn:

  ```sh
  $ yarn install
  ```

* Dev mode:

  ```sh
  $ yarn start
  ```

* Build mode:

  ```sh
  $ yarn build
  ```
